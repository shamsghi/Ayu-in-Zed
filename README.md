A faithful port of the original Ayu family for Zed, including:

- Ayu Classic Dark
- Ayu Classic OLED
- Ayu Classic OLED Bordered
- Ayu Classic Dark Mirage
- Ayu Classic Mirage
- Ayu Classic Light

## Install locally (dev extension)

1. Download zip from Releases
2. Unzip it
3. Open Zed.
4. Run `zed: install dev extension` from the command palette.
5. Select the folder.
6. Run `theme selector: toggle` and choose an Ayu variant.
